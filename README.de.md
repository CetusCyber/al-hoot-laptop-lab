# 🐋 AL-HOOT – Integriertes Laptop-Labor

## 🧾 Kurzbeschreibung
* 🐋 **AL-HOOT** ist ein Projekt, das ein einzelnes Laptop in ein voll funktionsfähiges Labor verwandelt.
* 💻 Es vereint mehrere Betriebssysteme, Wartungstools und Testumgebungen.
* 🤖 Führt lokale KI-Modelle direkt auf dem Gerät aus.
* 🧠 Komplett durch Selbststudium und Eigeninitiative entwickelt.

---

## 🎯 Projektziel
* Schaffung einer realen Arbeitsumgebung auf einem einzigen Gerät.
* Nutzung als Plattform für:
    * 🧪 Experimente
    * 🛠️ Wartung
    * 📚 Lernen

### 🔗 Integrierte Ebenen:
* 🪟 Ein optimiertes Windows-System.
* 🧰 Wartungs- und Rettungstools.
* 🐧 Linux-Umgebung für Sicherheitstests.
* 📱 Android-Umgebung.
* 🤖 Lokale Künstliche Intelligenz.

---

## ⚙️ Hardware
* **Gerät:** Huawei Laptop
* **Prozessor:** Intel Core i7 Evo
* **Arbeitsspeicher:** 16GB RAM
* **Speicher:** 1TB SSD

---

## 💽 Partitionierung
* **Laufwerk C:** Windows 11 LTSC (optimiert mit **AtlasOS**).
* **Laufwerk V:** **Ventoy** mit der **MediCat**-Bibliothek für Systemrettung.

---

## 🧱 System-Komponenten
1. **Hauptsystem:** Windows 11 LTSC + AtlasOS für maximale Performance.
2. **Wartungsebene:** Ventoy & MediCat zum Booten, Scannen und Reparieren.
3. **Linux-Ebene:** WSL + **Kali Linux Everything** für Security-Labs.
4. **Android-Ebene:** WSA zum Testen von Apps und Netzwerk-Tools.
5. **Isolationsschicht:** Windows Sandbox für sicheres Testen von Dateien.
6. **Lokale KI:** **Ollama** (via WSL) für cloud-unabhängige KI-Modelle.
