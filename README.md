# 🐋 AL-HOOT (The Whale) – Integrated Laptop Lab

## 🧾 Overview
* 🐋 **AL-HOOT** is a project that transforms a single laptop into a comprehensive digital laboratory.
* 💻 It integrates multiple operating systems, maintenance tools, and testing environments.
* 🤖 Runs local AI models directly on the hardware.
* 🧠 Built entirely through self-learning and individual effort.

---

## 🎯 Project Objectives
* Create a real-world production and testing environment on a single machine.
* Serve as a multi-purpose platform for:
    * 🧪 **Experimentation**
    * 🛠️ **Maintenance**
    * 📚 **Continuous Learning**

### 🔗 Integrated Layers:
* 🪟 **Debloated Windows:** For core performance.
* 🧰 **Maintenance Tools:** For system rescue and diagnostics.
* 🐧 **Linux Environment:** For security and penetration testing.
* 📱 **Android Subsystem:** For mobile app testing.
* 🤖 **Local AI:** Privacy-focused intelligence.

> 💡 **Philosophy:** Proving that self-learning is sufficient to build a professional lab without traditional academic paths.

---

## ⚙️ Hardware Specifications
* **Device:** Huawei Laptop
* **Processor:** Intel Core i7 Evo
* **Memory:** 16GB RAM
* **Storage:** 1TB SSD

---

## 💽 Disk Partitioning
### 🪟 Partition (C:)
* **OS:** Windows 11 LTSC
* **Optimization:** Stripped and optimized using **AtlasOS**.

### 🧰 Partition (V:)
* **Tool:** Ventoy
* **Payload:** **MediCat** library for maintenance and system rescue.

---

## 🧱 Environment Components
1. **Primary OS:** Windows 11 LTSC + AtlasOS for peak performance.
2. **Rescue Layer:** Ventoy + MediCat for booting, scanning, and repair.
3. **Linux Layer:** WSL + **Kali Linux (Everything)** for security audits.
4. **Android Layer:** WSA (Windows Subsystem for Android) for network tools and app testing.
5. **Isolation Layer:** Windows Sandbox for safe execution of untrusted files.
6. **Local AI:** **Ollama** running via WSL with optimized local LLMs.
